Dog Tracker

GET

How to get list of all dogs:

URL/dogs

GET

How to get dog by Id:

URL/dogs/:id

POST

How to post a dog:

URL/dogs

Structure Example:

{ 
    "name": "Bambi", 
    "breed": "chihuahua", 
    "weight": 20, 
    "img" : Image URL 
}

PUT

How to put a dog:

URL/dogs/:id

Structure Example:

{ 
    "name": "Bambi", 
    "breed": "chihuahua", 
    "weight": 20, 
    "img" : Image URL 
}

DELETE

How to delete a dog by Id:

URL/dogs/:id
